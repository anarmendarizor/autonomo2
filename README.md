Este proyecto consiste en el desarrollo del juego clásico Snake, programado en Python, como parte del Proyecto Integrador de la asignatura Lógica de Programación. El objetivo principal del proyecto es aplicar los conceptos fundamentales de programación mediante la creación de un software funcional, estructurado y modular, evidenciando cómo la tecnología influye en el desarrollo de videojuegos y el entretenimiento digital.

El propósito del proyecto es desarrollar el juego Snake utilizando Python para aplicar la lógica de programación, el uso de estructuras condicionales y repetitivas, así como la implementación de funciones que permitan organizar el código de manera clara y eficiente. A través del proyecto, se refuerza la importancia de la programación como herramienta para la creación de soluciones tecnológicas y aplicaciones interactivas.

El desarrollo del juego se realizó utilizando Python 3 y el entorno de desarrollo Spyder, ejecutándose completamente en la consola. Para su funcionamiento se emplearon únicamente librerías estándar como os para la limpieza de pantalla, time para el control del tiempo y random para la generación de la comida, garantizando la compatibilidad del programa sin depender de librerías externas.

El proyecto está organizado en dos archivos principales: un archivo principal que contiene el menú y el control general del programa, y un archivo de funciones que concentra toda la lógica del juego Snake, como el movimiento de la serpiente, la generación de la comida, el control del crecimiento y la detección de colisiones. Esta separación permite un código más limpio, reutilizable y fácil de mantener.

El juego permite al usuario controlar la serpiente mediante las teclas W, A, S y D, incrementar el puntaje al consumir la comida representada por el carácter X, y finalizar la partida al colisionar con los bordes del tablero o consigo misma. Al finalizar el juego, se muestra el puntaje final y el usuario puede regresar al menú principal para iniciar una nueva partida o salir del programa.

Este proyecto integra los contenidos de las cuatro unidades de la materia, aplicando algoritmos, diagramas de flujo, estructuras de control y programación funcional. Como parte de la entrega final, el proyecto se encuentra alojado en un repositorio de GitHub e incluye la documentación correspondiente y un video demostrativo que muestra el funcionamiento completo del juego.
